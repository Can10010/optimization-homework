# 最优化理论与方法 —— 作业与实验代码汇总

本仓库用于整理《最优化理论与方法》课程相关的**作业代码**与**实验代码**。

代码内容严格对应课程作业与实验要求，  
不进行额外建模扩展或算法改写，仅用于完成课程作业与复现实验结果。


### 实验五：代码获取说明

实验五基于已有开源 / 教学脚本进行修改实现。

正常情况下，**仅根据日志即可完成实验**，不需要额外代码。

来信中请注明以下信息（用于身份确认）：

- 所在学校  
- 本课程上课的教学楼

> 非本课程同学，或未按日志尝试修改者，恕不提供代码。
