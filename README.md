# 最优化理论与方法 —— 作业与实验代码汇总

本仓库用于整理《最优化理论与方法》课程相关的**作业代码**与**实验代码**。

代码内容严格对应课程作业与实验要求，  
不进行额外建模扩展或算法改写，仅用于完成课程作业与复现实验结果。

---

## 目录结构说明

```

.
├─ 作业/      # 课程作业代码（含部分实验型作业）
├─ 实验/      # 课程实验代码与实验日志
└─ README.md

```

---

## 作业部分

作业相关代码与说明位于：

👉 **[作业目录 README](./作业/README.md)**

内容包括：

- 各次作业对应代码
- 作业中涉及的实验内容
- 每次作业的代码结构与可调参数说明

---

## 实验部分

实验相关代码与实验日志位于：

👉 **[实验目录 README](./实验/README.md)**

内容包括：

- 每个实验对应的代码文件
- 实验日志（记录题目、实现思路、参数位置）
- 不涉及算法原理推导，仅用于实验复现

---

## 说明

- 本仓库**以完成课程作业为目标**
- 不作为算法库或教学示例仓库
- 代码结构以“清晰、可复用、可修改”为优先

如课程要求或题目发生调整，可直接根据各目录 README 中的说明修改对应代码。
