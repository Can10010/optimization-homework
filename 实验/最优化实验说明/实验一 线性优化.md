## 水泥调运问题

### 题目（原题意简述）

有 m 个水泥厂 A_i，年产量为 a_i；  
有 k 个城市 B_j，年需求量为 b_j；  
单位运输成本为 c_ij，且满足：

∑ a_i = ∑ b_j

目标是在满足供需的前提下，使总运输成本最小。

---

### 我是怎么对应到代码的

- 决策变量  
  x_ij：水泥厂 i 向城市 j 运的数量  
  → 代码里的 `X`（m × k 矩阵）

- 目标函数  
  最小化 ∑ c_ij · x_ij  
  → `linprog(c_vec, ...)`

- 约束  
  - 行和 = a（供给）
  - 列和 = b（需求）
  - x_ij ≥ 0

---

### 代码里真正需要关注 / 能改的地方

```python
m, k = 10, 15        # 水泥厂数、城市数
a, b = ...           # 供给、需求
c = ...              # 成本矩阵
```

## 无约束优化

---




### 实验内容

本实验主要处理无约束优化问题，通过迭代方法求函数极小值。

---

### 题目（原题意简述）

给定目标函数 f(x)，通过迭代方法求其最小值点。

实验要求：
- 给定初始点
- 设置步长或搜索策略
- 比较不同参数下的收敛情况

---

### 我是怎么对应到代码的

- 决策变量  
  x：自变量（向量或标量）  
  → 代码中的 `x` / `x0`

- 目标函数  
  f(x)  
  → 代码中的 `func(x)`

- 梯度  
  ∇f(x)  
  → 代码中的 `grad(x)`

- 迭代过程  
  x_{k+1} = x_k - α_k ∇f(x_k)

---

### 代码里真正需要关心 / 可以改的地方

```python
x0 = ...          # 初始点
alpha = ...       # 步长（或步长策略）
max_iter = ...    # 最大迭代次数
tol = ...         # 收敛阈值
```
