# 实验四：智能优化算法实验

本实验使用 **PSO（粒子群）** 和 **GA（遗传算法）** 求解同一个无约束优化问题。  
仅记录：**题目 → 代码 → 可改参数 → 备注**。

---

## 一、实验题目

使用 PSO、GA 算法求解如下问题：

$$
\min z = 4x_1^2 + 5x_2^2 - 3x_1 - 2x_2 + 7
$$

$$
-4 \le x_1, x_2 \le 10
$$

## 三、算法说明（只保留必要）

### 1. PSO（粒子群）

- 决策变量：  
  $x = (x_1, x_2)$

- 更新方式：  
  速度 + 位置迭代

- 边界处理：  
  **截断法（clamp）**  
  若变量越界，直接拉回到上下界

---

### 2. GA（遗传算法）

- 编码方式：  
  实数编码

- 选择：  
  锦标赛选择（tournament）

- 交叉：  
  实数混合交叉（blend crossover）

- 变异：  
  高斯扰动

- 边界处理：  
  **截断法（clamp）**

---

## 四、实际可改参数（最重要）

### PSO 中：

```python
LB, UB = -4.0, 10.0      # 变量边界
N = 40                  # 粒子数
MAX_IT = 200            # 迭代次数

w = 0.72                # 惯性权重
c1 = 1.49               # 个体学习因子
c2 = 1.49               # 群体学习因子
````

---

### GA 中：

```python
LB, UB = -4.0, 10.0     # 变量边界
POP = 60                # 种群规模
GEN = 200               # 进化代数

PC = 0.9                # 交叉概率
PM = 0.2                # 变异概率
SIGMA = 0.4             # 变异强度
```

老师改题，**基本只动这些**。

---

## 五、结果说明

* 两种算法均能收敛到同一最优解附近
* 理论最优解为：

$$
x_1^* = \frac{3}{8}, \quad x_2^* = \frac{1}{5}
$$

* 数值结果允许存在轻微随机误差
